# SAR - الخطوط الحديدية السعودية

مشروع Next.js 15 لموقع الخطوط الحديدية السعودية

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

## البناء

```bash
npm run build
```

## البنية

- `src/app/` - صفحات Next.js App Router
- `src/components/` - مكونات React
- `src/app/globals.css` - الأنماط العامة
